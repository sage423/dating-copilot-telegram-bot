# AI Dating Copilot

Upload a Bumble/Tinder screenshot to Telegram.

The bot:
- Extracts profile/chat context
- Detects conversation phase
- Suggests next move
- Generates 3 reply options

Built using:
- n8n
- Claude Vision
- Telegram Bot API
